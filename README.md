# 💫 About Me:
I am a Frontend Developer with experience in building responsive, scalable, and user-friendly web applications. I specialize in **React.js, Next.js, TypeScript, Tailwind CSS**, and modern UI libraries like **Material-UI, Ant Design, and ShadcnUI**. I have worked on a variety of projects including **dashboards, LMS platforms, CRM systems, e-commerce sites, streaming platforms, and AI-integrated applications**.<br><br>I enjoy solving complex problems through clean code and intuitive design. I’m experienced in collaborating with cross-functional teams, conducting code reviews, and working with modern tools like **Redux, Zustand, React Query, Jest, Docker, and CI/CD**. I’m currently open to **Frontend Developer** opportunities where I can contribute to meaningful products and continue growing professionally.
